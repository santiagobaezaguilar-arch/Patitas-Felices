# 🐾 Patitas Felices - Maqueta Web

## 📌 Descripción

Este proyecto es una **maqueta web estática** para la fundación *Patitas Felices*, enfocada en mostrar su labor de rescate y adopción de animales.

La web está desarrollada únicamente con:

* **HTML5**
* **CSS3 (sin JavaScript)**

Incluye animaciones, diseño responsive y componentes interactivos simulados usando solo CSS.

---

## 🧩 Páginas del proyecto

### 🏠 `index.html`

Página principal del sitio.

Incluye:

* **Header con navegación** hacia adopción, donaciones, misión/visión y contacto 
* **Sección hero con carrusel animado (CSS)** usando `@keyframes` 
* Botones CTA:

  * ADOPTA
  * CONTÁCTANOS 
* **Testimonios** en formato de tarjetas
* **Catálogo de animales** (grid responsive con 8 ejemplos)
* **Misión y visión** en bloques separados
* **Carrusel secundario animado** con imágenes dinámicas 
* **Formulario de contacto básico**
* **Footer con redes sociales y branding**

---

### 🐶 `Adopta.html`

Página dedicada al proceso de adopción.

Incluye:

* Explicación del proceso de adopción 
* Requisitos:

  * Mayor de edad
  * Espacio adecuado
  * Compromiso con el animal 
* Enlace a formulario externo (Google Forms)
* Información de contacto
* Tarjeta destacada de animal

---

### 💰 `Donaciones.html`

Página enfocada en apoyo a la fundación.

Incluye:

* Información sobre el impacto de las donaciones 
* Tipos de ayuda:

  * Económica
  * Insumos
  * Voluntariado 
* Métodos de pago:

  * PayPal
  * Nequi 
* Tarjeta visual de apoyo

---

### 🎨 `style.css`

Archivo principal de estilos.

Características clave:

* Uso de **variables CSS (`:root`)** para colores y diseño 
* Diseño con:

  * Flexbox
  * Grid
* Animaciones:

  * Carruseles con `@keyframes`
  * Efecto shimmer en header
  * Hover effects en botones y tarjetas 
* Diseño responsive con **media queries**
* Tipografía personalizada (`@font-face`)

* `@media (max-width: 900px)`
* `@media (max-width: 600px)`

## ⚠️ Limitaciones

* No hay backend ni base de datos
* Formularios no son funcionales (solo visuales)
* No hay JavaScript (todo es CSS puro)

---

## 💡 Detalles destacables (lo que te suma puntos)

* Paleta de colores bien estructurada con variables
